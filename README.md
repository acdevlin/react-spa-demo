# react-spa-demo
Sample demo of a Single Page App (SPA) using React + Vite, including basic navigation and routing.

It is hosted on Github Pages at https://acdevlin.github.io/react-spa-demo/

Created via the following tutorial: https://brendan-munnelly.github.io/full-stack/react/spa-intro.html


## Build & Deployment
* Building: `npm run build`
* Running locally: `npm run dev`
* Deployment to github pages: `npm run deploy -- -m "your commit message here"`
